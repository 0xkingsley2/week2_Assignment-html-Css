Digital Skills Training Academy Website
========================================

Project Overview:
-----------------
This is a multi-page educational website for a Digital Skills Training Academy.
The website demonstrates HTML structure, hyperlinks, images, and advanced CSS styling
including Flexbox, CSS Grid, and responsive design.

Folder Structure:
----------------
academy-website/
│
├── index.html          (Home Page)
├── about.html          (About Page)
├── courses.html        (Courses Page)
├── contact.html        (Contact Page)
├── css/
│   └── style.css       (Main stylesheet)
│
├── images/             (Image folder - currently using external images)
│
└── README.txt          (This file)

Pages Description:
------------------

1. index.html (Home Page)
   - Website logo with emoji icon
   - Navigation menu linking to all pages
   - Hero section with background image
   - Welcome message
   - Featured courses section (3 courses)
   - Footer with links and contact info

2. about.html (About Page)
   - Information about the academy
   - 2 images (classroom scenes)
   - Mission statement
   - Vision statement
   - Company values
   - Achievements section

3. courses.html (Courses Page)
   - 8 courses offered
   - Images for each course
   - Brief description of each course
   - Duration information
   - Why learn with us section

4. contact.html (Contact Page)
   - Contact information (address, phone, email, hours)
   - Contact form with fields for name, email, phone, subject, message
   - Social media links (Facebook, Twitter, Instagram, LinkedIn, YouTube)
   - FAQ section

HTML Features:
--------------
- Semantic HTML tags: <header>, <nav>, <main>, <section>, <article>, <footer>
- Hyperlinks between all pages
- Links to 3 external websites (W3Schools, MDN Web Docs, freeCodeCamp)
- 10+ images throughout the website with appropriate alt attributes
- Different image sizes used appropriately

CSS Features:
-------------
- Custom colors (CSS variables for easy theming)
- Background images (hero section with gradient overlay)
- Borders on cards and sections
- Box shadows for depth
- Rounded corners (border-radius)
- Flexbox for navigation and layouts
- CSS Grid for course cards and responsive layouts
- Responsive design with media queries for:
  * Mobile devices (max-width: 768px)
  * Tablets (769px - 1024px)
  * Desktop computers (1025px+)

How to Use:
-----------
1. Open index.html in a web browser to view the home page
2. Navigate between pages using the navigation menu
3. All pages are linked together correctly
4. The website is fully responsive and works on all devices

Note:
-----
- The contact form is a visual demonstration and does not send real emails
- Social media links are placeholders and should be updated with actual URLs

Technologies Used:
------------------
- HTML5
- CSS3 (Flexbox, CSS Grid, Custom Properties, Media Queries)
- No JavaScript required for basic functionality


Browser Compatibility:
---------------------
- Modern browsers (Chrome, Firefox, Safari, Edge)
- Responsive design works on mobile, tablet, and desktop
